# WiFi Network Dashboard

لوحة تحكم إدارة شبكات WiFi - Web Dashboard

## المتطلبات
- Node.js 18+

## التثبيت

```bash
npm install
```

## الإعداد

1. انسخ `.env` وعدّل `VITE_API_URL`

## التشغيل

```bash
# Development
npm run dev

# Production Build
npm run build
```

## الـ Stack
- React 19
- Vite
- Tailwind CSS v4
- React Query
- Zustand
- Recharts
- Leaflet (React-Leaflet)
- Axios

## النشر

```bash
# Vercel
vercel --prod
```
